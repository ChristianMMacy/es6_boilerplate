## Getting Started

#### Install dependencies:
```
$ npm i
```
#### Run karma
```
$ npm t // runs tests via PhantomJS 1.9.8
<!-- or -->
$ npm run debug // for debugging in Chrome
```
#### Coverage 
- Open coverage dir and see report.
